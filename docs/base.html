Class Helper Untuk Memperbesar Ukuran Huruf

Selain heading <h1> hingga <h6>tersedia .class helper untuk memperbesar ukuran huruf. Class ini dapat digunakan bersama heading (<h1> hingga <h6>), Paragraph (<p>).
.class tersebut antara lain :

Giga Class

Mega Class

Kilo Size

Alpha Size

Beta Size
