Class Helper Untuk Memperbesar Ukuran Huruf

Selain heading <h1> hingga <h6>tersedia .class helper untuk memperbesar ukuran huruf. Class ini dapat digunakan bersama heading (<h1> hingga <h6>), Paragraph (<p>).
.class tersebut antara lain :

Giga Class

Mega Class

Kilo Size

Alpha Size

Beta Size

Gamma Size

Delta Size

Epsilon Size

Zeta Class

HTML Code

<p class="giga">Giga Class</p>
<p class="mega">Mega Class</p>
<p class="kilo">Kilo Size</p>
<p class="alpha">Alpha Size</p>
<p class="beta">Beta Size</p>
<p class="gamma">Gamma Size</p>
<p class="delta">Delta Size</p>
<p class="epsilon">Epsilon Size</p>
<p class="zeta">Zeta Class</p>
