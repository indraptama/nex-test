Javascript

Modal

Modal adalah javascript untuk memunculkan content baru secara full diatas halaman page (dengan menghitamkan halaman disekelilingnya). Contoh Modal adalah image berukuran besar yang muncul apabila meng-klik image thumbnails dalam halaman gallery

Cara untuk dapat memunculkan modal adalah dengan menuliskan link menuju image berukuran besar, dan menambahkan class="has-modal" pada link anchor (<a>) tersebut.

HTML Markup

<a href="img/gallery/gallery-ori/cherry-red-ori-malang.jpg" class="has-modal">
	<img src="img/gallery/gallery-thumbnail/cherry-red-ori-malang-thumb.jpg" alt="">	
</a>

Modal dapat memuat caption dengan menambahkan title="foo barrr (isi dari caption)" setelah class="has-modal".

HTML Markup

<a href="img/gallery/gallery-ori/cherry-red-ori-malang.jpg" class="has-modal" title="Cherry Red Original (Malang) - Isi Caption Modal">
	<img src="img/gallery/gallery-thumbnail/cherry-red-ori-malang-thumb.jpg" alt="">	
</a>

Background Changer

Background Changer adalah javascript untuk menambah background yang dapat berubah secara dinamik pada suatu element tertentu. Contoh penggunaan background changer adalah hero unit pada halaman utama

Background Changer diatur dengan javascript yang ditempatkan pada bagian akhir halaman sebelum </body>

HTML Markup
<script>
$(".hero").backstretch([
	"img/slider/slide-1.jpg", //lokasi image 
	"img/slider/slide-2.jpg", //lokasi image 
	"img/slider/slide-4.jpg", //lokasi image
	], 
	{
		fade: 750, 
		duration: 4000 //mengatur lama pergantian
	});
</script>
</body>

Tooltip

Tooltip adalah javascrpit untuk memunculkan ballon kecil berisikan text ketika menghover suatu element. Contoh dari tooltips adalah ballon yang muncul ketika menghover color selection di halaman product

Untuk memunculkan tooltip adalah menambah class="tooltip" disertai title="(text-tooltip)" pada element yang di hendaki

ceri
HTML Markup
<a href="#dock1-ceri" class="img--round color-thumb-small cherry-red tooltip" title="Cherry Red">ceri</a>

Tabs

Tabs adalah javascript yang berfungsi untuk menyembunyikan sebagian content, dan memunculkan content tersebut apabila link anchor di klik. Contoh tabs adalah side-navigation di halaman about

Untuk membuat tabs dibutuhkan link (<a href="">) menunju #id suatu element yang dimuat dalam suatu container div, ul dan nav dengan class="idTabs". Element #id yang telah dibuat link achor dalam class="idTabs secara otomoatis akan disembunyikan

OWL Slider

Owl Slider adalah content slider yang dapat bergerak secara automatis ataupun secara manual. Contoh Owl slider adalah gambar pada bagian our gallery di halaman home (index.html)

Untuk membuat owl slider dibutuhkan beberapa content yang disimpan dalam suatu kontainer ul atau div dengan diberi id="owl-slider" dan class="swipe".

Owl slider diatur menggunakan javascript dalam main.js. Secara default owl slider akan menampilkan 4 gambar yang akan berubah jumlahnya menjadi lebih sedikit sesuai dengan ukuran screen.

Javascript Markup
$(document).ready(function() {
	$("#owl-slider").owlCarousel({
    	autoplay: 3000, //Untuk mengatur lama perpindahan
    	goToFirst: true,
    	goToFirstSpeed:  1000,
    	navigation: true, // memunculkan navigasi
    	items: 4, // mengatur jumlah item dalam sekali tampilan
	});
});
